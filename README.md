# MXN442 Assignment 2: Modern Computing Techniques

**Data Science/Machine Learning Paper Analysis and Implementation**

## Overview

This repository contains the work for MXN442 Assignment 2, which involves analyzing and implementing a post-2020 peer-reviewed data science/machine learning paper focused on regression or classification tasks.

## Assignment Requirements

### Paper Selection Criteria
- **Publication Date**: Post-2020
- **Type**: Peer-reviewed research paper
- **Domain**: Data Science/Machine Learning (regression or classification)
- **Venue**: Preferably Q1 venues (top-tier conferences/journals)
- **Data Availability**: Must include accessible datasets

### Deliverables

#### 1. Presentation
- **Schedule**: Week 12 or 13
- **Duration**: Maximum 10 minutes
- **Format**: In-class presentation

#### 2. Report
- **Length**: Maximum 10 pages
- **Submission**: Canvas by **October 17, 2025, 23:59**
- **Format**: Academic report with the following structure

### Report Structure

#### Required Sections

1. **Introduction**
   - Background and motivation
   - Research question and objectives
   - Paper overview

2. **Data**
   - Dataset description
   - Data characteristics and statistics
   - Data preprocessing steps

3. **Methods**
   - **Original Paper Methods**: Description of the methods used in the selected paper
   - **MXN442 Methods**: Implementation of three specific methods:
     - **Preprocessing**: Data cleaning, feature engineering, normalization, etc.
     - **Model**: Machine learning model implementation and architecture
     - **Cross-Validation (CV)**: Validation strategy and hyperparameter tuning
     - **Evaluation**: Performance metrics and evaluation methodology

4. **Results**
   - Reproduction of original paper results
   - Comparison with original findings
   - Performance analysis

5. **Discussion**
   - Analysis of results vs. original paper
   - Strengths and limitations
   - **GenAI Extension**: 
     - Integration of Generative AI tools/techniques
     - Include prompts used and analysis performed
     - Discussion of GenAI contribution to the work

6. **Conclusion**
   - Summary of findings
   - Future work and implications

### Code Requirements
- **Implementation**: Include all code used for the analysis
- **Format**: Can be provided as direct code files or links to repositories
- **Documentation**: Well-commented and reproducible code
